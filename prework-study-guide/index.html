Prework Study Guide

HTML

  • head: contains information about the webpage
  • body: represents the visible content shown to the user

CSS

  • margin: indicates how much space we want around the outside of an element
  • padding: indicates how much space we want around the content inside an element

Git

  • git status: checks what branch you are in
  • git add -A: adds changes for commits later
  • git commit -m 'write changes here': commits changes to main
  • git push origin main: pushes commits to main branch
  • git status: checks what branch we are currently on
  • git checkout main: checks if in main branch, switches to if not
  • git checkout -b branch-name: creates a new branch and switches to it
